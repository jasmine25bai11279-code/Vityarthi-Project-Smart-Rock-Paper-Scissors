Project Statement
Problem: Traditional RPS is random. Can we build an intelligent opponent using only logic programming and knowledge representation in Prolog?

Scope: Implement a complete RPS system in SWI-Prolog with rule-based AI that learns player tendencies using dynamic predicates – directly applying CO6 (Prolog Programming) and CO2 (Knowledge Representation).

Target Users: Students learning AI, Logic Programming enthusiasts

High-level Features:

Interactive gameplay with input validation
Rule-based intelligent opponent
Dynamic learning from game history
Strategy adaptation using backtracking and facts
